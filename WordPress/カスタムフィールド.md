# カスタムフィールド

- 「新規投稿を追加」→「オプション」→「カスタムフィールド」に✔︎

- single.phpに記述

- `$price = get_post_meta(get_the_ID(), '価格', true)`

- `$published = get_post_meta(get_the_ID(), '発売日', true)`

- `<?php echo $___; ?>`で出力する

## Advanced Custom Fields

- テンプレートタグ

`<?php the_field('  '); ?>`

- `<?php echo get_field('価格'); ?>円`

にすると、値を取得する

- `<?php echo number_format(get_field('価格')); ?>円`

にすると、3桁ごとにカンマが入る
